🛒 E-Commerce Sales Dashboard

📌 Project Overview
The **E-Commerce Sales Dashboard** is an interactive Power BI project designed to analyze and visualize e-commerce sales data. It provides valuable business insights into sales performance, profitability, customer behavior, product categories, and regional trends through dynamic dashboards and interactive visualizations.



🚀 Features
- 📈 Interactive Sales & Profit Dashboard
- 💰 Key Performance Indicators (KPIs)
  - Total Sales
  - Total Profit
  - Total Orders
  - Average Order Value
- 🛍️ Category & Sub-Category Performance Analysis
- 🌍 State-wise Sales and Profit Analysis
- 👥 Customer-wise Sales Insights
- 📅 Monthly Sales & Profit Trend Analysis
- 🎛️ Interactive Filters and Slicers
- 📊 Dynamic Charts and Visualizations



🛠️ Tools & Technologies
- **Power BI**
- **Microsoft Excel**
- **Power Query**
- **DAX (Data Analysis Expressions)**



📂 Dataset
The dashboard uses an e-commerce sales dataset containing information such as:
- Order ID
- Customer Name
- Product Category
- Sub-Category
- Sales
- Profit
- Quantity
- State
- Payment Mode
- Order Date


📊 Dashboard Insights
- Identifies top-performing product categories and sub-categories.
- Tracks monthly sales and profit trends.
- Compares sales and profitability across different states.
- Analyzes customer purchasing patterns.
- Highlights high-performing products and business regions.
- Monitors overall business performance using interactive KPIs.


🎯 Business Objective
The objective of this dashboard is to help businesses monitor sales performance, identify profitable products and regions, understand customer purchasing behavior, and support data-driven decision-making.
